ERD folder
----------

Files:
- requirements.md       -> Task description & entity list
- sample_schema.sql     -> Example CREATE TABLE statements
- airbnb-erd.drawio     -> Placeholder; open in draw.io to create the visual ERD
- (export) airbnb-erd.png -> Export from draw.io and add here

How to edit:
1. Open ERD/airbnb-erd.drawio in https://app.diagrams.net/ (File → Open from Device).
2. Draw the ER diagram using entities, mark PK and FK, and export PNG.
3. Commit the PNG alongside the .drawio file.
