# Task 2 — Database Schema (DDL)

This folder contains the SQL DDL for the Airbnb database:

- `schema.sql` — CREATE TABLE statements with constraints, keys, and indexes.
- Tables include: User, Property, Address, Booking, Payment, Review, Amenity, PropertyAmenity, Photo.
- Indexes are added for optimized lookups on foreign keys.

All tables are designed based on the normalized database design (3NF).
